# Rhythm Chavda — Personal Portfolio

A modern, high-end personal portfolio website built with **Next.js** (App Router), **Tailwind CSS**, and **Framer Motion**.

## 🚀 Features

- ⚡ Next.js 16 with App Router + Static Export
- 🎨 Glassmorphism UI with dark/light mode toggle
- 🖱️ Custom cursor with trailing ring animation
- 📜 Scroll progress indicator
- ✨ Framer Motion animations (fade, slide, stagger)
- ⌨️ Typewriter role effect in Hero
- 📱 Fully responsive (mobile-first)
- 🔍 SEO optimized (meta tags, OpenGraph, Twitter cards)
- 💬 Contact form with client-side validation
- 🏆 Achievements timeline
- 💻 Projects showcase with animated cards
- 🛠️ Skills with animated progress bars
- 🔗 Social links (GitHub, LinkedIn, LeetCode, Unstop, TryHackMe)

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 🏃 Development

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

This outputs a static site to the `out/` directory for GitHub Pages deployment.
